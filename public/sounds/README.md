# Sound Files Attribution

This app uses sound effects. For production deployment, download free sounds from:

1. **Card Deal Sound** (`card-deal.mp3`)
   - Search: "card swoosh" or "card flip"
   - Source: https://freesound.org/

2. **Chip Clink Sound** (`chip-clink.mp3`)
   - Search: "poker chips" or "casino chips"
   - Source: https://freesound.org/

3. **Win Sound** (`win.mp3`)
   - Search: "cash register" or "cha-ching"
   - Source: https://freesound.org/

4. **Loss Sound** (`loss.mp3`)
   - Search: "buzzer" or "wrong answer"
   - Source: https://freesound.org/

5. **Casino Ambience** (`casino-ambience.mp3`)
   - Search: "casino ambience" or "casino background"
   - Source: https://freesound.org/

## Instructions

1. Visit https://freesound.org/
2. Search for each sound type
3. Download MP3 files (choose Creative Commons licensed sounds)
4. Rename files to match the names above
5. Place all files in the `/public/sounds/` directory

## License Requirements

- Ensure you comply with Creative Commons licenses
- Provide attribution if required by the specific sound licenses
- Keep this file updated with proper attribution links
